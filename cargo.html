Заполните разделы по итогам недели. Отчёт скачается на ваше устройство после нажатия «Завершить заполнение» — его нужно передать для сводки.

Отправка на Google Диск
Вставьте URL веб-приложения Apps Script (см. инструкцию, которую прислал Claude). После сохранения каждый «Завершить заполнение» будет отправлять .html-отчёт в папку на Диске — сводка читает данные из него же.
