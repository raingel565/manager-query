Отдел продаж · Сводный отчёт

Сводка по отчётам менеджеров

Загрузите .html-отчёты, которые менеджеры скачали из опросника (или отправили на Google Диск). Сводка соберёт данные по разделам автоматически — в том числе по разделам, добавленным вручную.

1. Загрузка отчётов

Перетащите файлы сюда или выберите вручную. Можно добавлять отчёты по одному — старые не пропадают.
Перетащите .html-отчёты сюда, или нажмите, чтобы выбрать (старые .json тоже подойдут)

2. Фильтры

Диапазон дат применяется к дате заполнения отчёта. Менеджеров можно исключить из сводки, сняв отметку.
